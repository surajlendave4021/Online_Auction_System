# Online Auction System

## Description

The Online Auction System is a web-based application developed using Python Django. It allows sellers to upload products for auction and buyers to place bids online.

## Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* Bootstrap
* SQLite

## Features

* User Registration
* User Login
* Seller Module
* Buyer Module
* Admin Panel
* Product Management
* Online Bidding
* Auction Winner Selection
* Report Generation
